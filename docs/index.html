GPShike

GPS hiking is an open source android app for hiking.

Its MainActivity calls 8 Activities through a 4X2 imagebutton menu.
GPShike through this activities utilizes:

App Demonstration

1 DatumActivity.java

With this activity the user can define a Datum and a projection.

To define a datum the user must enter

  • Ellipsoid parameters (name, semi-major axis, inverse flattening)
  • 7-parameter transformation (translations, rotations, scale)
  • Projection parameters (center ö0, ë0, scale factor)

2. MapsActivity.java

Initializes the map resources. Uses Osmdroid free maps.

  • Uses standard MAPNIK tiles (free worldwide maps)
  • Good coverage for Greece and Europe
  • No API keys required

Basic Map Controls:

  • Zoom in/out buttons
  • "My Location" button
  • Multi-touch gestures supported

Functionality for Other Activities:

  • Methods to add waypoints (addWaypointMarker)
  • Methods to draw tracks (drawTrack)
  • Methods to center map (centerMapOnPoint)

Permissions Handling:

  • Properly requests location permissions
  • Gracefully handles permission denial

Performance Considerations:

  • Map caching for offline use
  • Efficient overlay management

3. WaypointsActivity.java

Record waypoints when managing map background.

Waypoints are recorded in lat, lon coordinates.

Integrated Map View:

  • Shows all waypoints as markers
  • Real-time updates when recording/editing
  • Zoom controls (+/- buttons)
  • Center-on-location button

Waypoint Recording:

  • 5-character uppercase ID validation
  • Automatic numbering
  • GPS coordinates capture
  • Timestamp recording

Editing Interface:

  • Full list view of all waypoints
  • Add new waypoints
  • Delete all functionality
  • Automatic map refresh after editing

File Management:

  • CSV format with all fields
  • Persistent storage in app directory
  • Automatic loading on startup

User Experience:

  • Clear control layout
  • Immediate visual feedback
  • Error handling and toasts
  • Smooth transitions between views

4. TracksActivity.java

Trackpoints are recorded also in lat long csv files.

Track Recording:

  • Start/Stop recording button
  • 5-second interval recording
  • 5-character track ID validation
  • CSV format with all required fields
  • Automatic numbering of points

Map Visualization:

  • Real-time track drawing
  • Different colors for different tracks
  • Auto-pan when points go off-screen
  • Zoom controls (+/- buttons)
  • Center-on-location button

Editing Interface:

  • Full list view of all track points
  • Add new points
  • Delete all functionality
  • Automatic map refresh after editing

File Management:

  • Persistent storage in app directory
  • Automatic loading on startup
  • Proper error handling

User Experience:

  • Clear status indicators
  • Immediate visual feedback
  • Smooth transitions between views
  • Proper handling of GPS signal loss

5. ImportActivity.java

Import Functionality:

  • Uses Android's file picker to select CSV files
  • Validates file selection
  • Copies selected file to either waypoints_cache.txt or tracks_cache.txt
  • Provides status feedback

User Interface:

  • Simple three-button layout
  • Status text view for feedback
  • Consistent design with other activities
  • Clear button labels

File Handling:

  • Uses Apache Commons IO for reliable file operations
  • Proper error handling
  • Supports both waypoints and tracks
  • Maintains original file formats

6. ExportActivity.java

Export Functionality:

  • Uses Android's share dialog to export files
  • Checks if data exists before exporting
  • Uses FileProvider for secure file sharing
  • Provides status feedback

7. SatelitesActivity.java

Sky Plot Visualization:

  • Uses radar chart to display satellite positions
  • Center of chart represents zenith (90° elevation)
  • Outer edge represents horizon (0° elevation)
  • Directions marked (N, NE, E, SE, S, SW, W, NW)

Satellite Grouping:

  • Different colors for each constellation (GPS, GLONASS, Galileo, BeiDou)
  • Shows PRN numbers in tooltips

Information Display:

  • Shows satellite count per constellation
  • Clear visualization of satellite distribution

User Interface:

  • Simple layout with just a back button
  • Full-screen chart for maximum visibility
  • Clean, professional appearance

8. TranslateActivity.java

File Selection:

  • Uses Android's file picker to select input CSV files
  • Handles both WGS84 (ö,ë,h) and projected (E,N,H) outputs

Coordinate Transformations:

  • Converts WGS84 to Cartesian (XYZ) using ellipsoid parameters
  • Optionally projects to Mercator (ENH) if projection defined
  • Handles 7-parameter datum transformations

Output Files:

  • Creates .XYZ file with Cartesian coordinates
  • Creates .ENH file with projected coordinates (if projection defined)
  • Preserves original point IDs and descriptions

Parameter Handling:

  • Loads ellipsoid and projection parameters
  • Validates required parameters before transformation

User Interface:

  • Simple three-button interface
  • Status feedback during operations
  • Error handling and warnings
